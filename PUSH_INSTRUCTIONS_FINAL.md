## Troubleshooting

**"Permission denied (publickey)"** → SSH issue, ignore it. Use HTTPS instead.

**"401 Unauthorized"** → Credential/token issue:
   - Verify token is correct (no spaces)
   - Regenerate new token at https://github.com/settings/tokens

**"fatal: Could not read from remote repository"** → Network issue:
   - Check internet connection
   - Try pushing with: `git push -u origin claude/api-key-env-priority-Wwjia --verbose`
